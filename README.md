# **Teamboard – Collaborative Workspace & Task Board (Laravel + SvelteKit)**

Teamboard is a modern Trello-like project management system built with:

- **Laravel 12** – Backend REST API  
- **SvelteKit + TypeScript** – Interactive frontend UI  
- **SQLite** – Zero-config local database  
- **Modular architecture** – Clean data models & controllers  
- **Reactive state** – Fast UI with drag & drop support  

This project showcases strong **full-stack architecture**, clean REST API design, and modern UI engineering.

---

## ✨ Features

### **Backend (Laravel 12)**
- Workspaces, Boards, Lists, and Cards REST API  
- Migrations + fully relational models  
- Eloquent relationships (1-N, N-N)  
- Structured controllers following REST conventions  
- SQLite for instant development  
- Clear API endpoints  

### **Frontend (SvelteKit + TypeScript)**
- Sidebar workspace navigation  
- Board selection panel  
- Full Kanban-style board UI  
- Add List  
- Add Card  
- Delete Card  
- Drag & Drop card movement (client-side)  
- Clean component logic  
- Centralized API layer
